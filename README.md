# tackle-license

Tackle provider to generate a license file in a directory. 

## Usage 

Within a **`tackle.yaml`**:
```yaml
project_slug->: input 
license_>: tackle robcxyz/tackle-license --output {{project_slug}}
```

From command line:
```shell
tackle robcxyz/tackle-license
```
